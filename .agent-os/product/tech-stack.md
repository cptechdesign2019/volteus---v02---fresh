# Technical Stack

> Last Updated: 2025-07-29
> Version: 1.0.0

This document outlines the technical architecture and key technology choices for the Volteus project. It is based on the Agent OS standards and the project's PRD.

| Layer | Choice | Version / Notes |
| :--- | :--- | :--- |
| **Application Framework** | Next.js | v15+ (App Router) |
| **Database System** | Supabase (PostgreSQL) | Multi-tenant via RLS |
| **JavaScript Framework** | React | v19+ |
| **CSS Framework** | Tailwind CSS | v4.0+ |
| **UI Component Library** | shadcn/ui | Radix UI primitives |
| **Fonts Provider** | Google Fonts (self-hosted) | Montserrat |
| **Icon Library** | Lucide React | - |
| **Application Hosting** | Render.com | - |
| **Database Hosting** | Supabase Cloud | - |
| **Asset Hosting** | Supabase Storage | - |
| **Deployment Solution** | GitHub Actions to Render | - |
| **Code Repository URL** | [volteus-v02-fresh_07.29.25](https://github.com/cptechdesign2019/volteus---v02---fresh) | - |
| **AI Integration** | Google Gemini | For Scope of Work generation |
| **Email Service** | MailerSend | For sending quotes and notifications |
